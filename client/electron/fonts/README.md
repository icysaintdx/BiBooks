# 公文字体配置

## 需要的字体文件

根据《党政机关公文格式》(GB/T 9704-2012) 要求，需要以下字体：

| 字体名称 | 文件名 | 用途 | 说明 |
|---------|--------|------|------|
| 方正小标宋简体 | FZXiaoBiaoSong-B05S.ttf | 标题 | 公文标题专用 |
| 方正仿宋简体 | FZFangSong-Z02S.ttf | 正文 | 正文标准字体 |
| 方正楷体简体 | FZKai-Z03S.ttf | 附件/注释 | 附件说明、批注 |
| 方正黑体简体 | FZHei-B01S.ttf | 小标题 | 一级标题、重点内容 |

## 获取字体

1. 从方正字库官网购买商业授权
2. 或使用开源替代字体（见下方）

## 开源替代方案

如果无法获取方正字体，可使用以下开源字体：

| 开源字体 | 文件名 | 替代 |
|---------|--------|------|
| 思源宋体 | SourceHanSerifSC-Regular.ttf | 方正小标宋/仿宋 |
| 思源黑体 | SourceHanSansSC-Regular.ttf | 方正黑体 |
| LXGW WenKai | LXGWWenKai-Regular.ttf | 方正楷体 |

## 字体安装

将字体文件（.ttf 或 .otf）放入此目录，程序会自动加载。
