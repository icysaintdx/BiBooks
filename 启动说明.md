# BiBooks 启动说明

## 快速启动

### Windows 用户

**方式一：双击启动（推荐）**
- 双击 `start.bat` 文件即可

**方式二：PowerShell 启动**
```powershell
.\start.ps1
```

**方式三：命令行启动**
```cmd
cd client
npm install  # 首次运行需要
npm run dev
```

## 系统要求

- **Node.js**: 16.0 或更高版本
- **npm**: 7.0 或更高版本
- **操作系统**: Windows 10/11

## 首次运行

1. 双击 `start.bat` 或运行 `.\start.ps1`
2. 系统会自动检测并安装依赖（约 2-5 分钟）
3. 依赖安装完成后，应用会自动启动

## 常见问题

### Q: 提示"未检测到 Node.js"
A: 请先安装 Node.js，下载地址：https://nodejs.org/

### Q: 依赖安装失败
A: 可能是网络问题，尝试：
1. 使用国内镜像：`npm config set registry https://registry.npmmirror.com`
2. 清除缓存：`npm cache clean --force`
3. 重新运行启动脚本

### Q: 启动后白屏
A: 等待几秒钟，Electron 应用需要加载资源

### Q: 端口被占用
A: 关闭占用 5173 端口的程序，或修改 `client/vite.config.ts` 中的端口配置

## 开发模式

启动后会自动打开开发者工具，可以：
- 查看控制台日志
- 调试前端代码
- 测试 IPC 通信

## 构建打包

```cmd
cd client
npm run build
npm run dist
```

打包后的安装包在 `client/dist` 目录下。
