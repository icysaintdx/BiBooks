# 核心模块功能说明

**日期**: 2026-06-05 20:00
**责任人**: BiBooks 开发团队
**版本**: v0.2.0

## 一、背景/问题描述

BiBooks 基于 OpenBidKit 二次开发，需要集成 BiaoShu-SKILL 的行业检测和 AI 去痕能力，并新增报价计算和报价脱敏功能，以满足自动标书生成系统的完整需求。

## 二、解决方案

新增 4 个核心服务模块，全部采用 Node.js (CommonJS) 编写，与现有 Electron 架构完全兼容：

### 2.1 行业检测模块 (industryDetector.cjs)

**功能**: 自动识别招标文件所属行业，为目录生成和内容生成提供行业上下文。

**检测方法**: 基于关键词匹配，统计招标文件文本中各行业关键词的出现比例，选择置信度最高的行业。

**支持行业**:
| 行业代码 | 行业名称 | 关键词示例 |
|----------|----------|-----------|
| it_information | IT/信息化 | 信息化、软件、系统集成、云计算 |
| construction | 建筑工程 | 建筑、施工、工程、市政 |
| medical | 医疗健康 | 医疗、健康、医院、设备 |
| education | 教育服务 | 教育、学校、教学、培训 |
| manufacturing | 制造业 | 制造、设备、自动化、工业 |
| logistics | 物流运输 | 物流、运输、仓储、配送 |
| consulting | 咨询服务 | 咨询、管理、审计、法律 |
| general | 通用行业 | (兜底) |

**调用方式**:
```javascript
const { detectIndustry } = require('./industryDetector.cjs');
const result = detectIndustry(tenderText);
// result: { industryCode: 'it_information', confidence: 0.625, config: {...} }
```

### 2.2 AI 去痕模块 (humanizer.cjs)

**功能**: 去除 AI 生成文本中的典型痕迹，使内容更自然、更像人类书写。

**处理流程**:
1. **AI 特征词汇替换**: 赋能→支持、助力→帮助、驱动→推动 等
2. **填充短语删除**: 删除"首先，""众所周知，"等段落开头填充词
3. **长句拆分**: 超过 100 字的句子在逗号处断句
4. **重复表达变化**: 同一词出现超过 3 次时，替换部分为同义词

**调用方式**:
```javascript
const { humanize, humanizeChapters } = require('./humanizer.cjs');

// 单文本去痕
const { text, changes } = humanize(aiGeneratedText);

// 批量章节去痕
const { chapters, totalChanges } = humanizeChapters(chapterList);
```

### 2.3 报价计算模块 (pricingService.cjs)

**功能**: 管理投标报价明细、计算汇总、生成报价表格。数据纯本地存储，不经过任何外部服务。

**核心能力**:
- 报价单创建与管理（项目关联）
- 报价明细项的增删改查
- 自动计算小计、税额、优惠、合计
- 按分类分组汇总
- 金额数字转中文大写
- 生成 Markdown 格式报价表格

**数据结构**:
```javascript
{
  id: '唯一ID',
  projectName: '项目名称',
  currency: 'CNY',
  taxRate: 0.13,        // 增值税率
  discountRate: 0,       // 优惠率
  items: [
    {
      category: '设备费',
      name: '服务器',
      specification: 'Dell R740',
      unit: '台',
      quantity: 2,
      unitPrice: 50000,
      subtotal: 100000   // 自动计算
    }
  ],
  notes: '备注说明'
}
```

**调用方式**:
```javascript
const pricing = require('./pricingService.cjs');

const sheet = pricing.createPricingSheet({ projectName: 'XX项目' });
pricing.addPricingItem(sheet, { category: '设备费', name: '服务器', quantity: 2, unitPrice: 50000 });
const summary = pricing.calculatePricingSummary(sheet);
const markdown = pricing.generatePricingTableMarkdown(sheet);
```

### 2.4 报价脱敏中间件 (desensitization.cjs)

**功能**: 在调用 AI 之前将报价/金额等敏感数据替换为占位符，返回后还原。确保企业报价数据永远不会泄露到云端 AI。

**匹配模式**:
- 货币符号金额: ¥123,456.78
- 带单位金额: 100万元、50,000元
- 关键词后金额: 报价：123,456.78
- 百分比: 13%
- 大额数字: 123,456

**脱敏流程**:
```
原始文本: "本项目报价 ¥150,000.00，税率13%"
    ↓ 脱敏
脱敏文本: "本项目报价 {{BIBOOKS_CURRENCY_SYMBOL_1}}，税率{{BIBOOKS_PERCENTAGE_2}}"
    ↓ 发送给 AI
AI 返回: "根据报价 {{BIBOOKS_CURRENCY_SYMBOL_1}}..."
    ↓ 还原
最终结果: "根据报价 ¥150,000.00..."
```

**调用方式**:
```javascript
const { desensitize, resensitize, withDesensitization } = require('./desensitization.cjs');

// 方式1: 手动脱敏/还原
const { text, mapping } = desensitize(originalText);
const aiResult = await callAI(text);
const finalResult = resensitize(aiResult, mapping);

// 方式2: 中间件模式（推荐）
const finalResult = await withDesensitization(originalText, async (desensitizedText) => {
  return await callAI(desensitizedText);
});
```

## 三、代码示例

完整的标书生成流程（集成新模块后）:

```javascript
const { detectIndustry } = require('./industryDetector.cjs');
const { humanizeChapters } = require('./humanizer.cjs');
const { withDesensitization } = require('./desensitization.cjs');
const pricing = require('./pricingService.cjs');

// 1. 解析招标文件后，检测行业
const industry = detectIndustry(tenderText);
console.log(`检测到行业: ${industry.config.name}`);

// 2. 生成内容时，使用脱敏中间件
const content = await withDesensitization(tenderText, async (safeText) => {
  return await aiGenerateContent(safeText, industry);
});

// 3. 内容去痕
const { chapters: polishedChapters } = humanizeChapters(generatedChapters);

// 4. 生成报价表
const sheet = pricing.createPricingSheet({ projectName: bidName });
// ... 添加报价项 ...
const pricingTable = pricing.generatePricingTableMarkdown(sheet);
```
