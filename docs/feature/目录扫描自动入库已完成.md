# 目录扫描自动入库已完成

## 日期
2026-06-22 12:00

## 责任人
开发团队

## 背景 / 问题描述
用户希望在知识库和企业知识库中新增「选择目录自动入库」功能：选择本地目录后，递归扫描所有子目录、解析所有支持格式文件（文档、图片、Office 等），用 AI 识别内容并自动入库，遇到同名文件时提示用户选择是否跳过，并发限制 3 个。

## 解决方案

### 后端层

#### aiService.cjs — Vision 图片分析回退
新增 `analyzeImageWithVision(imagePath, fileName)` 函数：读取图片文件转 Base64，构建多模态消息（image_url + text），调用文本模型的 Vision 能力识别图片内容为 Markdown。限制图片 ≤ 5 MB。

#### knowledgeBaseService.cjs — 扩展格式 + 扫描 + 批量导入
- `supportedExtensions` 从 6 种扩展至 18 种（新增 `.txt .ppt .pptx .xlsx .xls .png .jpg .jpeg .bmp .tif .tiff .webp`）
- `IMAGE_EXTENSIONS_SET` 常量用于图片判断
- `scanDirectoryRecursive(dirPath, maxFiles=500)` 递归扫描目录，上限 500 个文件
- `prepareDocument` 新增 Vision 回退：解析失败 + 文件是图片时调用 `aiService.analyzeImageWithVision` 获取 Markdown 文本
- 新增 `previewDirectoryScan(folderId)` — 打开目录选择器，扫描文件列表，标注同名文件，返回预览数据
- 新增 `importFromDirectory(folderId, filePaths, webContents)` — 并发 3 路批量创建文档记录并调用 `prepareDocument`

#### privateKnowledgeBase.cjs — 企业知识库目录扫描
- 工厂函数扩展为 `{ db, app, aiService, configStore }`
- 新增 `scanAndImportDirectory(webContents)` — 打开目录选择器，递归扫描，解析每个文件为 Markdown，AI 分类（company/team/cases/winning_bids/templates/reusable_blocks），调用 `createItem` 写入数据库，推送 `private-kb:event` 进度事件

#### IPC + Preload
- `knowledgeBaseIpc.cjs` 新增 `knowledge-base:preview-directory-scan`、`knowledge-base:import-from-directory`
- `privateKnowledgeBaseIpc.cjs` 新增 `private-kb:scan-and-import-directory`
- `preload.cjs` 暴露 `knowledgeBase.previewDirectoryScan`、`knowledgeBase.importFromDirectory`、`privateKnowledgeBase.scanAndImportDirectory`、`privateKnowledgeBase.onEvent`

### 前端层

#### KnowledgeBasePage.tsx — 两步向导 Dialog
- 新增 `scanDialog` 状态（step: preview/importing/done）
- `handleScanDirectory()` 调用 `previewDirectoryScan` 获取预览数据
- `handleStartScanImport()` 过滤重复文件后调用 `importFromDirectory`
- 工具栏新增"扫描目录"按钮
- 三步 Dialog：预览（文件数、重复标注、跳过选项）→ 导入中（进度提示）→ 完成（成功/失败统计）

#### PrivateKnowledgeBasePage.tsx
- 新增 `scanLoading` 状态
- `handleScanAndImport()` 调用 `scanAndImportDirectory`，完成后刷新列表和统计
- 工具栏新增"批量导入目录"按钮
- 订阅 `private-kb:event` 进度事件，在扫描完成后自动刷新

#### ipc.ts — 类型声明更新
- `knowledgeBase` 接口新增 `previewDirectoryScan`、`importFromDirectory`
- `privateKnowledgeBase` 接口新增 `scanAndImportDirectory`、`onEvent`

## 代码示例

```javascript
// knowledgeBaseService.cjs - 扫描目录预览
async previewDirectoryScan(folderId) {
  const result = await dialog.showOpenDialog({ title: '选择要批量导入的目录', properties: ['openDirectory'] });
  if (result.canceled) return { canceled: true };
  const allFiles = scanDirectoryRecursive(result.filePaths[0]);
  const { documents } = knowledgeBaseStore.list();
  const folderDocNames = new Set(documents.filter((d) => d.folder_id === folderId).map((d) => d.file_name));
  const files = allFiles.map((filePath) => ({ filePath, fileName: path.basename(filePath), isDuplicate: folderDocNames.has(path.basename(filePath)) }));
  return { canceled: false, dirPath: result.filePaths[0], files, totalCount: files.length, duplicateCount: files.filter(f => f.isDuplicate).length };
}
```
